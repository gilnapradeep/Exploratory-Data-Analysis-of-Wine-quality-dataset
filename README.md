# Wine Quality EDA

This project performs Exploratory Data Analysis (EDA) on the Wine Quality dataset.

## Objectives
- Understand dataset structure
- Find missing values
- Analyze feature distributions
- Detect correlations
- Identify factors affecting wine quality

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Files
- wine_quality_check.ipynb
- WineQT.csv

## Output
The project reveals important relationships between:
- alcohol
- acidity
- sulphates
- quality score
